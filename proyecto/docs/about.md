# Acerca de este proyecto

Este sitio ha sido creado como parte de una práctica de documentación utilizando **MkDocs** y el tema **Material for MkDocs**.  
El objetivo es aprender a generar documentación clara, estructurada y fácil de mantener utilizando herramientas modernas y profesionales.

---

## ¿Qué contiene este proyecto?

- **Páginas informativas** escritas en formato Markdown.
- Una **navegación sencilla** configurada desde `mkdocs.yml`.
- Un entorno de desarrollo servido mediante **Docker**, permitiendo ver los cambios en tiempo real.
- La posibilidad de **publicar** la documentación automáticamente en GitHub Pages.

---

## Tecnologías utilizadas

- **MkDocs**: generador de documentación estática.
- **Material for MkDocs**: tema moderno y muy utilizado en documentación técnica.
- **Docker**: para ejecutar MkDocs sin instalar dependencias en el sistema.
- **GitHub Pages** (opcional): para el despliegue del sitio web.

---

## Autor del proyecto

Este sitio ha sido desarrollado como parte de una práctica académica, con enfoque en:

- Aprender a estructurar documentación profesional.
- Comprender el funcionamiento de MkDocs y su flujo de trabajo.
- Practicar CI/CD con GitHub Actions (opcional).

---

Si quieres ampliar el proyecto, añadir nuevas páginas o aplicar configuraciones más avanzadas, es tan sencillo como editar los archivos en `docs/` y actualizar el archivo `mkdocs.yml`.
